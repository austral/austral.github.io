## If Statement

Examples:

```
if test() then
    doSomething();
end if;
```

```
if test() then
    doSomething();
else
    doSomethingElse();
end if;
```

```
if a() then
    doA();
else if b() then
    doB();
else if c() then
    doB()
else
    doElse();
end if;
```

Examples:


## While Loop

Examples

```
-- An infinite loop
while true do
    doForever();
end while;
```
